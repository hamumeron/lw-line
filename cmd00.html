lw line cmd--

  • lw grant … 初回に実行してPCのフォルダアクセスを許可
  • lw neW_FOLDA \name="フォルダ名"\ … 新しいフォルダを作成
  • lw neW_FILE \name="ファイル名"\ … 新しいファイルを作成
  • lw writE FILE \name="ファイル名"\ … 書き込みモードにして内容入力
  • lw FILE \name="ファイル名"\ … ファイルの中身を参照
  • lw deletE_FILE \name="ファイル名"\ … 指定したファイルを削除
  • lw Device_IP … あなたのグローバルIPを表示
  • lw connect https://example.com … 指定したURLにバックグラウンド接続
  • lw ruN_JavaScript \script="console.log('Hello');"\...新しいタブ（about:blank）を開き、指定JavaScriptを実行※Javascript部分はHTMLでも可

コマンドはすべて lw から始める必要があります。

※ name="" の指定時には必ず前後に \ を付けてください。
